# Приложение-резюме

Технологии, применённые для проекта:

- React JS ( + React Router DOM )
- Javascript
- HTML ( Valid, Semantic )
- CSS ( CSS Modules )



Сайт можно посмотреть/покликать на GitHub Pages:

https://samolevich.github.io/cv/



### Запуск локально

`npm install`

`npm start`
